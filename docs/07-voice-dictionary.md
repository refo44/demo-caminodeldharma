# Camino del Dharma — Voice Dictionary

**Léxico del sistema**  
**Versión 1.0**

Define qué palabras existen dentro del territorio de la comunidad y cuáles no pueden existir jamás.

La voz y el tono se definen en `06-guia-voz-microcopy-ux`.

---

## 1. Principio rector

Si una palabra pertenece al lenguaje de:

- marketing
- ventas
- growth
- plataformas
- startups
- monetización agresiva

entonces no pertenece a este sitio.

El sitio no persuade. No empuja. No captura. Acompaña.

---

## 2. Verbos permitidos

| Verbo | Uso |
|-------|-----|
| Practicar | Acción principal |
| Participar | Meditación, eventos |
| Explorar | Secciones, contenido |
| Conocer | La comunidad, el linaje |
| Contactar | Comunicación |
| Enviar | Formularios |
| Volver | Navegación |
| Inscribirse | Eventos |
| Donar | Sostener la comunidad |

**Nunca usar:** comprar, obtener, desbloquear, acceder, registrarse, suscribirse, activar, descubrir, probar, unirse (si suena a marketing).

---

## 3. Sustantivos permitidos

| Categoría | Palabras |
|-----------|----------|
| Comunidad | comunidad, práctica, sangha |
| Buddhismo | Dharma, Buddhismo, Chan, Tierra Pura, Mahāyāna |
| Práctica | meditación, estudio consciente, recitación |
| Personas | persona, practicante, maestro |
| Actividad | encuentro, retiro, taller, celebración |

**Nunca usar:** producto, contenido, usuario, audiencia, cliente, lead, funnel, plataforma, dashboard, feed.

---

## 4. Navegación

| Función | Término |
|---------|---------|
| Home | Inicio |
| Comunidad | La comunidad |
| Linaje | El linaje |
| Práctica | Práctica |
| Eventos | Eventos |
| Contacto | Contacto |

**Nunca:** dashboard, feed, profile, store, shop, portal.

---

## 5. Botones estándar

| Función | Texto |
|---------|-------|
| CTA principal | Practica con nosotros |
| Meditación | Participar |
| Evento | Inscribirme |
| Formulario | Enviar |
| Donaciones | Donar / Sostener la comunidad |

**Nunca:** Ver más, Comprar ahora, Empieza aquí, Suscríbete, Descubre, Únete (si suena a CTA agresivo).

---

## 6. Mensajes del sistema

| Situación | Texto |
|-----------|-------|
| Sin eventos | "No hay eventos programados en este momento." |
| Error envío | "No fue posible enviar el mensaje." |
| 404 | "Esta página no existe." |

**Nunca:** Error 404, No data, Loading, Oops, Algo salió mal.

---

## 7. Términos buddhistas

Usar con consistencia:

- **Buddhismo** (con doble 'd' según preferencia de la comunidad)
- **Dharma**
- **Chan**
- **Tierra Pura**
- **Mahāyāna**
- **Śākyamuni**
- **parinirvāṇa**
- **Vesak**

---

## 8. Regla final

Si una palabra suena como si pudiera aparecer en una landing, una app, una startup o una tienda, entonces no puede estar aquí.

El sitio no vende. El sitio acoge.
