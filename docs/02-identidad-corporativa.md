# Camino del Dharma — Identidad corporativa

**Versión 1.0 (borrador)**

Este documento define el sistema de identidad visual y tipográfica para el sitio de la Comunidad Buddhista Camino del Dharma. La Lluvia de ideas indica: **mantener paleta de colores actual** y **manual de marca (colores, fuentes, logos)**.

**Fuente canónica:** `content-source/Pagina web Camino del Dharma/Identidad CAMINO DEL DHARMA- (1).pdf`

---

## 1. Esencia de marca

Camino del Dharma no es una página comercial ni informativa. Es un espacio de acogida.

| Rasgo | Descripción |
|-------|-------------|
| Voz | Sobria, acogedora, clara |
| Estética | Calma, amplios espacios en blanco |
| Ritmo | Pausado, respirado |
| Función | Orientar, inspirar confianza, facilitar primer contacto con la práctica buddhista |
| Presencia | Comunidad como eje central |

El sitio debe transmitir **calma, claridad y coherencia**, priorizando la experiencia del usuario por encima del impacto visual inmediato.

Una vez definida, estructurar como:

- **Roles semánticos:** `--bg`, `--text`, `--text-muted`, `--surface`, `--link`, `--primary`, etc.
- **Accesibilidad:** contraste AA mínimo; consideración para discapacidad visual (Sandra tiene información completa)

## 3. Sistema tipográfico

**Pendiente:** Extraer del PDF de identidad.

**Recomendación:** Dos familias máximo. Una para títulos y contenido principal; otra para navegación, metadatos y formularios. Tipografías legibles, sin adornos.

---

## 4. Ritmo editorial

### Texto corriente

- Medida: ~65ch (ancho humano)
- Line height: 1.5–1.6
- Espacio: amplios márgenes, respiración

### Navegación y UI

- Discretos, sin dominar el contenido
- Sin CTAs agresivos

---

## 5. Gramática de layout

| Regla | Significado |
|-------|-------------|
| Una columna / layout simple | Lectura vertical |
| Espacio | Respiración, calma |
| Sin grids densos | No revista ni e-commerce |
| Flujo claro | Guiar hacia la práctica |
